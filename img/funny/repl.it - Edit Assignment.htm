To create an black layer over an image with jQuery, we need to use the hover event.

Using the toggleClass jQuery function, make the overlay background color 100% transparent when the mouse hover the #pictureDiv... And then set transparency back to 50% when the mouse stops hovering the #pictureDiv.

Hint:
Create a new class that updates the background transparency, and toggle it.
